Overview

This course will teach you to use Python to become a more effective and efficient developer, equipping you to solve the problems you face in your work. It is aimed at programmers who are towards the start of their career, and want to become better at using Python on their own and in a team.

The course will be taught through a mixture of demonstrations, exercises, and discussions. You'll spend a lot of time reading code and a lot of time writing code. Throughout the course, we'll have a strong focus on being useful and practical.

Learning outcomes

By the end of the course, you will have learnt how to:

  • structure your code so that it is maintainable in the future;
  • use Python idiomatically to help communicate with team members;
  • write effective tests to give you the confidence that your code is correct and that it remains correct as it develops; and
  • use tools to help you debug problems with your code.

Minimum experience

This course is for you if you're towards the beginning of your programming career. You should have a working understanding of functions, composite data types (such as lists and dicts), and iteration.
