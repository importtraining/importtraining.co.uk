Upcoming Courses

Python ~ Beyond the Basics

Friends House, London. nth-nth October 2015

This course is aimed at junior developers, graduates and people looking for new ideas / directions with Python.

We will be introducing and exploring intermediate level Python programming including, testing (using unittest and mock), tooling (pdb, pip, virtualenv), idiomatic Python and learning how to get the most out of the Python community.

By the end of this course you'll have the confidence to write well designed, maintainable and modular Python applications.
